# Changelog

## [Unreleased]

## [0.3.0] - 2026-07-01

### Added

- Session builder with category, mode, question-count, and timer choices
- Random, preliminary adaptive, weak, strong, and mixed card ordering
- Full-screen rapid-fire question experience
- Reverse-entry custom numeric keypad with hold-to-clear backspace
- Swipe right to skip, swipe left for don't know, and hold to reveal
- Accessible button alternatives for every gesture
- Instant, Correct, Slow, Incorrect, and Timeout classifications
- Session summary, accuracy, response time, and classification breakdown
- Per-card attempt statistics and locally saved session history

## [0.2.0] - 2026-07-01

### Added

- Eighteen default categories and a useful starter card library
- Custom category creation, editing, selection, and deletion
- Card creation, editing, duplication, deletion, favourites, and practice marks
- Question, answer, category, favourite, and practice filtering
- Pasted `question = answer` import with validation and preview
- Versioned LocalStorage data model with future learning-stat fields

## [0.1.0] - 2026-07-01

### Added

- Responsive app shell for iPhone and Windows
- Working Home, installation, and appearance screens
- Locally saved dark, light, and system themes
- Online/offline status
- PWA manifest and offline service worker
- Standard and maskable app icons
- Keyboard focus and reduced-motion support
- Product architecture and acceptance checks
